# Movie Night Bingo Card Generator
I love to watch movies with my partner. Often when we are watching we find there are reoccuring themes or inside jokes and decided to turn that into a game. This tool was built to take pictures we drew to represent those themes and inside jokes and randomize them onto a bingo card we can cross off as we watch. As things go, this game spiraled out into increasing complexity. 

## Prerequisites
You will need drawings or images to represent your themes or inside jokes. Included are some of ours to get an idea for how it all works.

## Getting Started
I recommend changing Complex Mode to False. This will bring the tool back to the basics of just a image shuffler.  

## Complex Mode
-Two images on one tile (positive, negative, and neutral combos)
-Background pattern
-Duplicate tiles or replaced tiles
-Places for extra points
-Luck out puts
-Image manipulate (reverse tiles)
